# Инструкция по развертыванию MakeBot на Yandex Cloud

## 1. Подготовка проекта

### Вариант A: Скачать готовый проект
1. Создайте папку `makebot-site` на своем компьютере
2. Скопируйте все файлы из структуры выше в соответствующие папки
3. Убедитесь, что структура соответствует примеру

### Вариант B: Создать файлы вручную
Вы можете скопировать код каждого файла и создать их вручную с помощью текстового редактора.

## 2. Локальная проверка (перед загрузкой на сервер)
